# File URLs

|                                      | Desktop | Mobile |
| ------------------------------------ | ------- | ------ |
| **[`require()`][require]**           | ✅      | ❌     |
| **[`requireAsync()`][requireAsync]** | ✅      | ✅     |

You can require files using file URLs:

```js
require('file:///C:/path/to/vault/then/to/script.js');
```
